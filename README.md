# AI Log Analyzer

A Python-based tool that uses AI (Groq LLaMA 3.3) to analyze system logs, detect suspicious activity, and generate security reports.

## Features

- CLI menu-based log analyzer
- AI-powered log analysis
- Real-time log monitoring
- Streamlit web dashboard (WIP)
- Report saving to JSON files

🚧 Work in progress. Code is being actively developed and improved.
